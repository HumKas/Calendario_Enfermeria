- API 

## Instalación

Clonar repositorio:

```bash
git clone https://github.com/HumKas/Calendario_Enfermeria.git
```

Crear entorno virtual:

```bash
python -m venv venv
```

Activar entorno:

Mac/Linux:

```bash
source venv/bin/activate
```

Instalar dependencias:

```bash
pip install -r requirements.txt
```

Ejecutar servidor:

```bash
python manage.py runserver
```

API:

```txt
/api/citas/
```
